# HLD → NC Liste Online App

Ovo je Streamlit web aplikacija za popunjavanje Vodafone/Telefonica NC liste iz HLD PDF-a.

## Kako koristiti online

1. Napravi GitHub račun ako ga nemaš.
2. Napravi novi GitHub repository, npr. `hld-nc-online`.
3. Uploadaj sve fajlove iz ovog ZIP-a u taj repository:
   - `app.py`
   - `hld_nc_automation.py`
   - `requirements.txt`
   - `.streamlit/config.toml`
4. Otvori Streamlit Community Cloud: https://share.streamlit.io/
5. Klikni **New app**.
6. Izaberi svoj GitHub repository.
7. Kao main file upiši: `app.py`.
8. Klikni **Deploy**.

Nakon toga dobiješ web link koji otvaraš iz browsera. Ne treba administrator na PC-u.

## Korištenje aplikacije

1. Upload HLD PDF.
2. Upload NC Excel template.
3. Klikni **Analiziraj PDF i generiraj NC listu**.
4. Provjeri prikazana polja.
5. Download popunjene NC liste.

## Važno

V1 je prototip. Uvijek provjeri:

- sheet `AUTO_Extrakt`
- popunjene redove u `NC- Zuordnungsliste`
- adrese, Node/FN oznake, HP/HSI i US vrijednosti

## Lokalno pokretanje, samo za test

Ako negdje imaš PC gdje možeš instalirati Python:

```bash
pip install -r requirements.txt
streamlit run app.py
```
